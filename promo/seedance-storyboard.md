# 众声 15秒宣传视频 — Seedance 2.0 分镜脚本

## 策略

Seedance 2.0 擅长从**静态图 → 动态视频**的转换。我们的方案：
- 用 poster-a.html 和 index.html 的关键画面截图作为参考图
- 产品本身是纯 UI（emoji 角色 + 深色背景），没有真人面孔，完美适配 Seedance 限制
- 15 秒 = 一个完整的情绪弧线：**好奇 → 冲突 → 震撼 → 品牌**

## 方案 A：单段 15 秒（推荐）

用 IMG-3（「我反对！」画面）作为**核心参考图**，让 Seedance 生成一段包含动态效果的完整视频。

### Seedance 输入

**参考图**: IMG-3（「我反对！」的 app 截图）
**参考音频**: gavel.mp3

**Prompt（直接复制给 Seedance）:**

```
@image1 作为画面主体风格参考。

生成一段15秒的暗色系科技风动态视频：

画面1（0-3秒）：纯黑背景中，金色渐变大字「众声」从中央缓缓浮现，带有微光粒子飘散效果。字体粗重有力。下方出现小字「知乎讨论理解地图」，左右有淡金色竖线装饰。

画面2（3-7秒）：快速切换到对阵画面。深紫黑背景上，左边出现橙红色卡片（内有沙发emoji和文字「躺平派」），右边出现蓝色卡片（内有愤怒emoji和文字「奋斗派」），中间金色大字「VS」有能量脉冲效果。两张卡片带有轻微的呼吸感浮动。

画面3（7-11秒）：画面剧烈震动！红色大字「我反对！」从中央爆裂弹出，带有放射状速度线从中心向四周扩散，文字有强烈的阴影和发光效果。整个画面闪烁一次白光后恢复。这是全片最高潮。

画面4（11-15秒）：画面恢复平静，黑底上缓缓浮现金色文字：「他们以为自己在反对彼此，其实在反对同一堵墙。」最后叠加品牌标识「众声 VOICES」和绿色数字「80%意见一致」。

整体风格：逆转裁判游戏 + 赛博朋克配色（深紫黑底色 #0e0818，红色 #ff3030，金色 #f8d56e，绿色 #50e0b0）。不要出现任何人脸。字体粗黑有冲击力。
```

## 方案 B：三段拼接（备选，控制力更强）

拆成 3 段分别生成，后期拼接。每段 5 秒。

### 段 1：悬念（5秒）

**参考图**: IMG-1（导航页截图）作为首帧
**Prompt:**
```
@image1 作为视频首帧。

从这个暗色系APP界面开始，镜头缓缓推进。画面中的文字和卡片元素带有微弱的呼吸感发光效果。3秒后画面开始加速推近，一道白光从右侧划过，暗示即将切换。

风格：赛博朋克，深紫黑配色，科技感UI界面。不要出现人脸。
```

### 段 2：冲突高潮（5秒）

**参考图**: IMG-2（VS对阵画面）作为首帧，IMG-3（我反对！）作为尾帧
**Prompt:**
```
@image1 作为视频首帧，@image2 作为视频尾帧。

从左右两个角色卡片对峙的画面开始，中间的VS文字在脉冲发光。画面氛围越来越紧张，屏幕轻微震动。突然画面剧烈晃动，红色大字「我反对！」爆炸般弹出，伴随放射状速度线和屏幕白光闪烁。

风格：逆转裁判游戏的法庭对决感，暗色系，夸张戏剧效果。不要出现人脸。
```

### 段 3：揭示 + 品牌（5秒）

**参考图**: IMG-4（共识卡片截图）作为尾帧
**Prompt:**
```
@image1 作为视频尾帧。

从纯黑画面开始，金色文字一行行浮现：「他们以为自己在反对彼此」换行「其实在反对同一堵墙。」文字两侧有金色竖线装饰。然后平滑过渡到最终的品牌画面：「众声 VOICES」标识 + 绿色高亮的「80% 意见一致」数据。画面安静而有力。

风格：暗色极简，金色点缀，科技感。不要出现人脸。
```

## 方案 C：图生视频 + 音乐节奏（如果有 BGM）

如果有合适的 BGM 音频文件，可以用 Seedance 的音视频同步功能：

**参考图**: IMG-5（完整海报）
**参考音频**: 一段 15 秒带节奏感的 BGM

**Prompt:**
```
@image1 作为画面内容参考，@audio1 作为音频节奏参考。

根据音频节奏，将这张海报的各个元素逐一动态呈现。在重拍处切换到下一个元素，在最强拍处让红色「我反对！」大字爆裂弹出。最终在尾拍时所有元素归位，还原为完整海报。

风格：暗色系，逆转裁判游戏感，粗黑字体，赛博朋克配色。不要出现人脸。
```

## 建议

1. **首选方案 A** — 最省事，一次生成 15 秒，让 Seedance 发挥创意
2. 如果方案 A 效果不理想，**退回方案 B** — 分段控制更精确
3. **关键是 IMG-3 这张截图的质量** — 「我反对！」的画面是整个视频的视觉锚点
4. 建议用 **手机竖屏比例 9:16** 生成，适配朋友圈 / 知乎 / 小红书传播
5. 生成后可以用剪映等工具叠加 gavel.mp3 音效，在 7 秒处（「我反对！」弹出时）同步播放
